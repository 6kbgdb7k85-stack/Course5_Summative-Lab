# Course 5 Summative Lab
This is Flatizon a dummy e-commerce site for the Course 5

# SETUP
- Download the repo
- run npm install
- run npm run server and npm run dev to run the application locally

# Features
- Home Page
- - Landing page for the site
- Shop Page
- - lists all products. Has a search field to search products by name
- - Products show a name, category, description and price
- Admin Page
- - Here you can add, edit or delete products
